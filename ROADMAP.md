# Roadmap

## Phase 0 — Foundation

Monorepo, architecture contracts, UI system, secure configuration, CI, error tracking, migrations.

## Phase 1 — Product shell

Landing, authenticated dashboard, URL intake, local/static report presentation, accessible loading/error states. This is the implementation starting now.

## Phase 2 — Exploration pipeline

FastAPI job API, Redis queue, isolated Playwright exploration, screenshots/evidence storage, SSE progress.

## Phase 3 — Intelligence reports

Feature/technology extractors, structured GPT-5 inference, evidence validation, search, exports, report persistence.

## Phase 4 — Team readiness

Organizations, RBAC, billing, audit logs, quotas, observability, reliability testing.

## Phase 5 — Advanced intelligence

Compare mode, architecture diff, authorization-aware analysis, API behavior analysis, security posture, Figma/GitHub inputs, system-design exports.
